FAQs

What does a report mean?

A report is one anonymous signal from the community. Counts are shown as reports, not facts.

What happens when a website is not listed?

We check DNS for the host. If it exists, we create an empty record so reports can start there.

Can one person spam reports?

Reports use fingerprinting and rate limits so one person cannot keep voting on the same host over and over.
